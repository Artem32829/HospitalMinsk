       Данное программное средство, разработано для учета и выдачи больничных листов пациентам поликлиники №38 г. Минска.
       При регистрации нового врача, необходимо ПОДКЛЮЧЕНИЕ к интернету. На ваш Email адрес будет отправлен код подтверждения, которое необходимо будет ввести в требуемую строку приложения.
       При регистрации пациента, важно соблюдать дату рождения пациента, так как данная поликлиника является взрослой, и регистрация возможна только для лиц, достигших 18 лет.
       При выдачи больничного листа соблюдайте все условия заполнения полей, при неправильном заполнении вам будет указана ошибка, которую потребуется исправить. Важно, при заполнении поля «Дата следующего посещения», НЕ ДОЛЖНА превышать 14 дней от даты заполнения формы.
       При закрытии больничного листа, требуется ввести только «Номер больничного листа» и поставить галочку «Закрыть больничный лист», дата закрытия листа будет автоматически поставлена.
       Данный больничный лист можно распечатать, для этого нужно выбрать ФИО пациента и нажать на кнопку «Печать», однако можно и сохранить данный лист в PDF файле.
       В окне «Справка» можно увидеть подробную информацию для правильности заполнения полей.
